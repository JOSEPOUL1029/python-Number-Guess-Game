Number Guessing Game

A simple Python game where you try to guess a random number between 1 and 10.

How to Run

python guessing_game.py

How to Play

1. The computer chooses a number from 1 to 10.
2. Enter your guess.
3. The game will tell you if the number is higher or lower.
4. Keep guessing until you get it right.
5. Choose whether to play again.
